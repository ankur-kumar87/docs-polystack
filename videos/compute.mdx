## Live Migrate an Instance

Move a running VM to another compute host via the Live Migrate action. Pick the target host from the list or leave it empty to let the scheduler decide. Available when the instance is Active or Paused.

## Enable vTPM and Secure Boot

Harden an instance at launch with a Virtual TPM and UEFI Secure Boot — for Windows 11, BitLocker, LUKS disk encryption, and measured boot. Configure both from the Create Instance wizard's System Config → Advanced Options. Requires Xloud KMS.
