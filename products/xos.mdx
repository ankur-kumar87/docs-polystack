XOS is a purpose-built Linux distribution, pre-configured and hardened for Polystack private cloud deployments. Every cluster node in an Ironcore, XPCI, or XHCI environment runs XOS as its base operating system.

<Card title="XOS — Polystack Operating System" icon="external-link" href="https://polystack.tech/xos" color="#197560" horizontal>
  Product details and datasheet on polystack.tech
</Card>

XOS ships with the complete Polystack runtime pre-installed — all required system services, container runtimes, storage drivers, and the XDeploy management agent. This ensures every node starts from a known, validated baseline. Security-hardened out of the box with immutable configuration, restricted SSH access, and pre-configured firewall policies.
