# Polystack Technologies

This is the documentation repository for Polystack Technologies.

It contains the source files, structure, and configuration used to build and maintain the Polystack documentation site.

## What This Repository Includes

- Product and platform documentation
- Service and deployment guides
- API reference content
- Integration and security documentation
- Shared assets and documentation configuration

## Local Development

To preview the documentation locally, install the Mintlify CLI:

```bash
npm install -g mint
```

Start the local development server from the repository root:

```bash
mint dev
```

The local preview will be available at `http://localhost:3000`.

## Publishing

Changes pushed to the default branch can be used for deployment through the configured documentation publishing workflow.

## Repository Purpose

This repo is maintained as the central docs source for Polystack Technologies.
