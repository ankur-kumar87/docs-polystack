## XAVS 2025.1 LTS

<Update label="March 2026" description="XAVS 2025.1 LTS — General Availability" tags={["Stable", "LTS"]}>
  Long-term support release for the Xloud Advanced Virtualization Suite.

  | Category | What's included |
  |----------|----------------|
  | Compute | Instance lifecycle improvements, scheduling enhancements, resource management |
  | Storage | Multi-tier storage support with automated data placement |
  | Networking | Enhanced network provisioning, security group management |
  | Security | Strengthened identity, access control, encryption capabilities |
  | Management | Updated resource optimization and operational monitoring |

  <Note>XAVS 2025.1 LTS receives security patches and critical bug fixes for **3 years** from release date.</Note>
</Update>

## XPCI 2025.1

<Update label="March 2026" description="XPCI 2025.1 — General Availability" tags={["Stable"]}>
  Complete private cloud infrastructure platform with all enterprise services.

  | Category | What's included |
  |----------|----------------|
  | Compute | Virtual machine management, live migration, vertical scaling |
  | Storage | Block storage, object storage, software-defined storage |
  | Networking | SDN, load balancing, DNS, floating IPs |
  | Identity | Multi-domain authentication, federation, RBAC |
  | Security | Key management, encryption, vTPM, compliance |
  | Operations | Monitoring, disaster recovery, cluster optimization |
  | Kubernetes | Container orchestration as a service |
  | High Availability | Instance HA with automatic failover |
</Update>

## XDeploy 2025.1-beta

<Update label="March 2026" description="XDeploy 2025.1-beta" tags={["Beta"]}>
  Deployment and lifecycle management tool for Xloud infrastructure.

  | Category | What's included |
  |----------|----------------|
  | Cluster deployment | Guided wizard for initial cluster setup and node provisioning |
  | Configuration | Centralized service configuration with validation and templating |
  | Lifecycle | Start, stop, upgrade, and reconfigure services across the cluster |
  | Licensing | Built-in activation and validation via [license.xloud.tech](https://license.xloud.tech) |

  <Warning>XDeploy 2025.1-beta is a pre-release version. Some features may change before general availability.</Warning>
</Update>

## XOS 3.1

<Update label="March 2026" description="XOS 3.1 — Stable Release" tags={["Stable"]}>
  Purpose-built operating system for Xloud cluster nodes.

  | Category | What's included |
  |----------|----------------|
  | Base OS | Updated kernel, system libraries, package repositories |
  | Pre-configured | All required runtime dependencies installed and configured |
  | Security | Hardened OS defaults, SSH configuration, system-level protections |
  | Deployment | Integrated tooling for cluster bootstrapping and node enrollment |

  <Tip>XOS is distributed as a bootable ISO. Existing nodes can be upgraded in-place using XDeploy.</Tip>
</Update>
