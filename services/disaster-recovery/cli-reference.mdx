## Overview

Disaster Recovery disaster recovery operations -- including protection plans, replication management,
failover, failback, and DR testing -- are managed exclusively through the **Disaster Recovery Dashboard**
web interface. There is no CLI for Disaster Recovery operations.

<Info>
  All DR operations are performed through the Disaster Recovery Dashboard. Navigate to
  **Disaster Recovery** in the main navigation to access protection plans,
  site management, failover controls, and compliance reports.
</Info>

---

## Available Operations via Dashboard

| Operation | Dashboard Location |
|-----------|-------------------|
| Create protection plans | **Disaster Recovery → Protection Plans → Create Plan** |
| Manage replication | **Disaster Recovery → Protection Plans → [Plan]** |
| Monitor replication lag | **Disaster Recovery → Protection Plans** — lag column |
| Register sites | **Disaster Recovery → Sites → Register Site** |
| Configure replication links | **Disaster Recovery → Sites → Replication Links** |
| Execute failover | **Disaster Recovery → Protection Plans → [Plan] → Failover** |
| Execute failback | **Disaster Recovery → Protection Plans → [Plan] → Failback** |
| Run DR tests | **Disaster Recovery → Protection Plans → [Plan] → Test Failover** |
| View test reports | **Disaster Recovery → Reports** |
| Generate compliance reports | **Disaster Recovery → Reports → RPO/RTO Compliance** |
| View audit log | **Disaster Recovery → Reports → Audit Log** |
| Configure automatic failover | **Disaster Recovery → Recovery Plans → [Plan] → Automatic Triggers** |
