## Overview

Polystack Software-Defined Storage (Software-Defined-Storage) is powered by Ceph. Use `ceph` and `rbd` CLI commands to manage the cluster, pools, block storage images, and object gateway.

<Note>
  **Prerequisites**
  - SSH access to a Ceph monitor or admin node
  - `ceph` and `rbd` CLI tools installed (`apt install ceph-common`)
  - Ceph keyring with admin permissions: `/etc/ceph/ceph.client.admin.keyring`
</Note>

---

## Cluster Health

<CodeGroup>
```bash title="Cluster health summary"
ceph health
ceph health detail
```

```bash title="Cluster status"
ceph status
ceph -s
```

```bash title="Monitor quorum"
ceph quorum_status --format json | jq .quorum_names
```

```bash title="OSD tree"
ceph osd tree
```

```bash title="Cluster usage"
ceph df
ceph df detail
```
</CodeGroup>

## Pools

<CodeGroup>
```bash title="List pools"
ceph osd pool ls
ceph osd pool ls detail
```

```bash title="Create replicated pool"
ceph osd pool create volumes 128 replicated
ceph osd pool set volumes size 3
```

```bash title="Create erasure-coded pool"
ceph osd pool create ec-data 64 erasure default
```

```bash title="Show pool stats"
ceph osd pool stats volumes
```

```bash title="Get pool parameter"
ceph osd pool get volumes size
ceph osd pool get volumes crush_rule
```

```bash title="Set pool parameter"
ceph osd pool set volumes size 3
```

```bash title="Delete pool"
ceph osd pool delete volumes volumes --yes-i-really-really-mean-it
```
</CodeGroup>

## OSDs

<CodeGroup>
```bash title="List OSDs"
ceph osd ls
```

```bash title="Show OSD stats"
ceph osd df
```

```bash title="Mark OSD out (before removal)"
ceph osd out osd.3
```

```bash title="Mark OSD down"
ceph osd down osd.3
```

```bash title="Remove OSD"
ceph osd purge osd.3 --yes-i-really-mean-it
```

```bash title="Reweight OSD"
ceph osd reweight osd.3 0.9
```
</CodeGroup>

## RBD (Block Storage Images)

<CodeGroup>
```bash title="List images in pool"
rbd ls volumes
```

```bash title="Create image"
rbd create --size 50G volumes/my-image
```

```bash title="Show image info"
rbd info volumes/my-image
```

```bash title="Resize image"
rbd resize --size 100G volumes/my-image
```

```bash title="Create snapshot"
rbd snap create volumes/my-image@snap-1
```

```bash title="List snapshots"
rbd snap ls volumes/my-image
```

```bash title="Delete snapshot"
rbd snap rm volumes/my-image@snap-1
```

```bash title="Delete image"
rbd rm volumes/my-image
```
</CodeGroup>

## Object Gateway (RGW)

<CodeGroup>
```bash title="List buckets (admin)"
radosgw-admin bucket list
```

```bash title="Show bucket stats"
radosgw-admin bucket stats --bucket my-bucket
```

```bash title="Create RGW user"
radosgw-admin user create \
  --uid my-user \
  --display-name "My User" \
  --email user@example.com
```

```bash title="Show user info and keys"
radosgw-admin user info --uid my-user
```
</CodeGroup>
